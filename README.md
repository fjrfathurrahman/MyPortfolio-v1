## Instalasi

1. Clone repositori ini
   ```sh
   git clone https://github.com/fjrfathurrahman/MyPortfolio-v1.git
2. Masuk ke direktori proyek
   ```sh
   cd first-portfolio
3. Instalasi dependencies
   ```sh
   npm install 
   npm run
